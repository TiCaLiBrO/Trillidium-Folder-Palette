# Trillidium Folder Palette
This project allows you to color your folders, allowing you to color-code them to your liking.

![Picture of all 39 folders in all of their glory](https://github.com/TiCaLiBrO/Trillidium-Folder-Palette/blob/main/TrillidiumPalette.png)

If you like what you see and want to color your folders in this way for better organization or for aesthetics, the instructions are below.

## Installation

### Download Folder Painter

First, go [here](https://folder-painter.en.softonic.com/?utm_source=SEM&utm_medium=paid&utm_campaign=EN_CA_GamesForWindows_downloads_tp&gad_source=1&gad_campaignid=22816290447&gclid=Cj0KCQjwrZTRBhDSARIsAHidYfcWmScEkQNYI3eTsphgWTRsVErhu9NY16Jzcd5yw52z9Y5UZ9YVy58aAi44EALw_wcB) to download **Folder Painter** by Sordum for Microsoft Windows.

Sordum has their own color palettes, which are nice, but none of them are as consistent as mine.
